# Lushy Logos

This directory contains the official logo assets for Lushy.

## Files
- **[lushy_logo.png](./lushy_logo.png)**: The primary logo usage. Safe to use on light backgrounds.
- **[icon.png](./icon.png)**: The official app icon.

## Usage Guidelines
- **Clear Space**: Always maintain a minimum clear space equal to the height of the "L" around the logo.
- **Scaling**: Ensure the logo is legible. Do not scale below 24px height for digital use.
- **Distortion**: Do not stretch, squash, or rotate the logo.
- **Colors**: Do not alter the brand pink color (`#FF4081`).

> [!TIP]
> For the app icon, please check with the design team (or user) for the latest version.
