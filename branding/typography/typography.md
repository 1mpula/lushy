# Lushy Typography

Our typography is designed to be **Playful**, **Bold**, and **Readable**. We pair a characterful heading font with a clean, modern sans-serif for body text.

## Primary Typeface (Headings)
**Font Name**: Outfit  
**Source**: [Google Fonts - Outfit](https://fonts.google.com/specimen/Outfit)  
**Usage**: Main headlines, section titles, buttons, big numbers.  
**Weights**: 
- **Bold (700)**: Primary headings
- **SemiBold (600)**: Subheadings, buttons

## Secondary Typeface (Body)
**Font Name**: Inter  
**Source**: [Google Fonts - Inter](https://fonts.google.com/specimen/Inter)  
**Usage**: Body text, inputs, navigation, long-form content.  
**Weights**:
- **Regular (400)**: Standard body text
- **Medium (500)**: Navigation links, labels
- **Bold (700)**: Emphasized inline text

## Fallback Stack
If web fonts fail to load, use the system font stack:
`font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Helvetica, Arial, sans-serif;`
